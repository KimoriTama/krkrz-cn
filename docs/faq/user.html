用户篇 FAQ

前言

关于游戏玩家可能遇到的问题的常见问题FAQ。


启动
Q.MFPlat.DLL或MF.dll未找到。
A.Windows N版本不包含与多媒体相关的功能，因此无法启动。
安装Media Feature Pack 。

音乐
Q.播放音乐时首先发出“噗噗”声
A.-wsrecreate=yes 可能会修复。
直接在命令行选项中指定，或在吉里吉里设置设为「Sound:DirectSound次缓冲区重新生成：始终重新生成」。

鼠标・输入
Q.即使转动滚轮消息也不前进，没有反应
A.设置-wheel可能会修复。。
尝试dinput和message，然后选择一个更好的方法。
直接在命令行选项中指定，或在吉里设置中设为「输入:鼠标滚轮检测方法：:DirectInput」或「输入:鼠标滚轮检测方法：窗口消息」。
Q.什么都没做，消息却自己前进。
A.如果连接了手柄，并且按钮一直处于按住状态，则重复按键可能会导致消息前进。
此外，如果您选择了“自动阅读”，则会自动继续阅读。（译注：废话）
