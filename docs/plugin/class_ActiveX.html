メンバ

コンストラクタ
ActiveX
メソッド
onCreate (ウインドウ生成完了後に呼び出されるイベント 初期生成時やフルスクリーン処理などでウインドウが生成された後に呼ばれます 再構築後は、登録されていたイベントはすべて破棄される他、その他もろもろが 初期化されるので、このイベント内で必要なものを再設定する必要があります。 )
onDestroy (ウインドウ破棄前に呼び出されるイベント フルスクリーン処理やオブジェクトの破棄などでウインドウが破棄される前に呼ばれます。 )
setExternalUI (拡張UI指定 IEコンポーネントでのみ意味を持ちます。現在の版では、これ呼び出すことで コンテキストメニューが抑制されます。 )
setPos (座標指定 )
setSize (サイズ指定 )
プロパティ
height (縦幅 )
isValidWindow (ウインドウが有効かどうか(読み込み専用) )
left (左端座標 )
top (上端座標 )
visible (表示状態 )
width (横幅 )
