メンバ

メソッド
getAsBitmap (クリップボードの内容をビットマップとしてレイヤから取得する )
hasFormat (指定のフォーマットがクリップボードに格納されているかチェックする )
setAsBitmap (レイヤの内容をビットマップとしてクリップボードにセットする。 吉里吉里以外のアプリへ引き渡す場合、α値は255にクリアされる。(事前にレイヤを合成してαを無くしておくことを推奨) 吉里吉里へ引き渡す場合、α値はそのまま受け継がれる。 )
setMultipleData (クリップボードに複数形式のデータを一括登録する データは複数の「データ形式:データ本体」の対を辞書で指定する。 対応しているデータは以下の通り - text: テキストデータ - tjs: TJS式 - bitmap: レイヤ )
プロパティ
asText (クリップボードにテキストを登録する、もしくは取り出す 取得に失敗したらvoidを返す。 )
asTJS (クリップボードにTJS式を登録する、もしくは取り出す 取得に失敗したらvoidを返す。 TJS式は、メンバー「type」(文字列)および「body」(型を問わない)を持った辞書で無ければならない。 それ以外の形式のデータを登録しようとすると登録が失敗する。 )
