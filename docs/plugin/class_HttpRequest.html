HttpRequest

メンバ

コンストラクタ
HttpRequest
メソッド
abort (現在実行中の送受信のキャンセル )
getAllResponseHeaders (すべての HTTPヘッダを取得する )
getResponseHeader (指定したHTTPヘッダを取得する )
getResponseText (レスポンスをテキストの形で取得 )
onProgress (データ送受信イベント )
onReadyStateChange (readyState が変化した場合のイベント )
open (指定したメソッドで指定URLにリクエストする ※常に非同期での呼び出しになります )
send (リクエストの送信。送信処理は非同期実行されます。 エラー時も例外は発生しませんので、readyState と status で判定してください。 )
sendStorage (リクエストの送信。送受信は非同期実行されます。 エラー時も例外は発生しませんので、readyState と status で判定してください。 )
setRequestHeader (送信時に送られるヘッダーを追加する )
プロパティ
contentLength (レスポンスの Content-Length ヘッダに無い場合はファイルダウンロード完了後にサイズ確定します。 )
contentType (レスポンスの Content-Type (エンコーディング指定は含まない) )
contentTypeEncoding (レスポンスの Content-Type のエンコード指定 ヘッダの指定が優先されますが、未指定、かつ受信データが text/html の場合は ファイル冒頭にある META 指定も参照します。 )
readyState (通信状態。読み込み専用 )
response (レスポンス。読み込み専用。 )
responseData (レスポンス｡読み込み専用 )
status (レスポンスの HTTPステータスコード。読み込み専用 )
statusText (レスポンスの HTTPステータスの文字列 )
