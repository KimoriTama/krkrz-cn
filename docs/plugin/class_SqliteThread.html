SqliteThread

メンバ

コンストラクタ
SqliteThread
メソッド
abort (実行処理を中止させます。 また記録されてる処理結果を破棄します。 )
onProgress (更新処理経過 )
onStateChange (状態変更 )
select (検索処理をバックグランドで実行する。 結果は内部に蓄積されて完了後に引き渡されます。 )
update (更新処理をバックグランドでトランザクション実行する。 )
プロパティ
errorCode (エラーコード 処理中での最後のエラーコードを返します )
progressUpdateCount (onProgress で呼び出す頻度指定 指定されてる数を取得するたびに呼び出しが行われる )
selectResult (select取得データ(配列) )
state (実行ステート )
