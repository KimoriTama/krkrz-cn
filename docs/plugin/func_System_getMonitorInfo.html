System.getMonitorInfo

機能/意味
モニタ情報を取得
タイプ
Systemクラスのメソッド
構文
getMonitorInfo()
引数
near trueなら対象モニタが無かった場合に最も近いモニタの情報を返す
x,y 座標指定
x,y,w,h 範囲指定
window ウィンドウ指定（Window オブジェクトを渡す）
戻り値
%[ name, primary, monitor:%[ x, y, w, h ], work:%[ x, y, w, h ] ];\nモニタの情報。辞書内容は getDisplayMonitors と同じ (ただし intersect 情報は無い)。\nnear=false 時でモニタが無い場合は void を返す。
説明
