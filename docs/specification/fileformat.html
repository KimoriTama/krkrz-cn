文件格式


文件格式

种类形式后缀名说明
静止画像 Windows 位图 (Bitmap) .bmp 仅支持未压缩格式。
对于32位位图，认为具有α通道。
JPEG .jpg
.jpeg		 可以指定读入精度。
可以设定的值包括 'high' (高), 'normal' (正常), 'low' (低) 。
默认为'normal'。
low是IDCT的定点（整数）AAN，无色差放大插值。
normal是IDCT的定点（整数）LLM，具有色差放大插值。
high为IDCT浮点AAN，色差扩大插值。
关于指定方法请参考启动选项 -jpegdec (JPEG图像编码精度)。
Portable Network Graphic (PNG) .png α通道和透明色指定对应。
TLG5 .tlg 吉里吉里独有的形式。
其特点是展开速度快（PNG的4倍速度左右）。
可用图像格式转换器
TLG6 .tlg 吉里吉里独有的形式。
压缩比PNG和TLG5高，但展开时间是TLG5的两倍。
不过，即使这样也是PNG的2倍速度，所以速度还是很快的。
可用图像格式转换器转换。
其他 .* 可通过Susie-plugin进行扩展。
音乐 WAVE .wav 支持未压缩的WAVE格式。
Ogg Vorbis .ogg 需要wuvorbis.dll插件
其他 .* 可通过插件进行扩展。
视频 MPEG I .mpg
.mpeg
.mpv
.m1v
.dat		 仅支持MPEG I。
不支持MPEG2或更高版本。
需要 krmovie.dll。
播放器安装Windows Media Player 6.4或更高版本、
或安装DirectX 7或更高版本。
Windows Media Video (WMV) .wmv 需要 krmovie.dll。
播放器必须安装Windows Media Player 9或更高版本。
AVI .avi 非官方支持。
需要要 krmovie.dll。
由于Codec等问题会无法播放的问题，不提供官方支持。
动画 ASD 文件 .asd 吉里吉里独有的形式。
请参阅AnimationLayer.tjs的注释或是吉里吉里2/KAG3的动画
归档文件 XP3 .xp3 吉里吉里独有的形式。
通过Releaser生成。
其他 .* 可通过插件进行扩展。
字体 渲染字体 .tft 吉里吉里独有的形式。
使用渲染字体数据创建工具生成
TrueType/OpenType 字体 .ttf
.otf		 非官方支持。
无需在系统上安装字体即可使用。
需要要addFont.dll。仅Windows2000/XP/Vista 支持。
