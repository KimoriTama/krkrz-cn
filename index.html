吉里吉里Z

关于吉里吉里Z

吉里吉里Z是吉里吉里2的后续版本。
使用吉里吉里Z可以制作2D游戏和应用程序。
它在许多商用ADV/视觉小说游戏中使用。
本程序使用了e-mote/えもふり。
不仅是游戏，可以用于各种各样的用途。

KAG3是使用吉里吉里Z开发ADV/视觉小说游戏的系统。
krwidgets 是一个用于使用吉里吉里Z创建GUI应用程序的控件工具包。

吉里吉里2在本页中间。

操作环境
  • CPU : Pentium III 800MHz以上 ( 推荐1GHz以上 )
  • OS : Windows Vista, 7, 8, 10
  • 内存: 128MB以上 ( 推荐1GB以上 )
  • VIDEO : 65536色 (TrueColor, 16bit) 以上 推荐( 16777216色 (FullColor, 32bit) )
  • 需要DirectX 9DirectX 9 或更高版本
  • 播放WMV 时需要 Windows Media Player 9 或更高版本
  • 播放H.264 时需要 Windows 7 或更高版本

系统要求会因使用的功能而异。这并不表示在吉里吉里Z中编写的所有软件都能在这种环境中流畅地运行。
请不要复制和粘贴上述操作环境，并由制作者确认和设置实际操作环境。

※上述内容和吉里吉里2相比有很大的变化，除了OS以外条件基本上是一样的。不过，考虑到目前的情况，吉里吉里2的运行环境显得过低，因此在此将其提升。
